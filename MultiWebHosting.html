Multi-Tier Web Hosting

PUBLISHED ON July, 2023.

The Project

In the implemented multi-tier web hosting setup, separate EC2 instances were deployed for different layers, including Apache Tomcat for web servers, RabbitMQ for messaging, and MySQL for database servers. SSL certificates were configured to enable HTTPS, while DNS services from Freenom and Route53 were utilized for managing domain names and routing. Additionally, an Application Load Balancer (ALB) was established to load balance traffic across the web server instances

Technologies Used

  • AWS
  • EC2
  • RDS
  • DNS
  • Linux
  • GitHub
