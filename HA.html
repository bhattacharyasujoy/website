HA setup for Databases

Company project

The Project

Implemented a High Availability (HA) setup for PostgreSQL on CentOS using PostgreSQL's clustering and load balancing tool, pgpool-II. Ensured data integrity and minimized downtime by employing a cluster with an odd number of servers to prevent split-brain scenarios. Utilized both synchronous and asynchronous replication methods for data redundancy and failover preparedness. Monitored the system health and initiated automatic failover using pgpool-II's watchdog feature.

Technologies Used

  • PostgreSQL
  • Pgpool-II
  • Watchdog
  • Heartbeat
  • Ubuntu/CentOS
